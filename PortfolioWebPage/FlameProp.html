Project
Flame Propagation

//Real-time simulation of flame propagation on voxelized 3D-geometry using Unity.

Inspired by the flame propagation mechanic found in the Far Cry -series, I decided to explore flame simulation in real-time computer graphics. Having familiarized myself with the Unity game engine, this was the choice platform to build the simulation platform.

By utilizing voxelization, I was able to create volumetric representation of any given 3D-object. In this particular simulation, voxels served as a method of mapping a ‘terrain’ for the flame to propagate. Sub-sectioning a mesh into voxel data is fast, and easily runs in real-time.

This project served as an excellent introduction to the realm of advanced computer simulations.

My mesh voxelization implementation extends Nihilist Dev’s blog post found here.

I was able to utilize this implementation for my simulation, by making a few minor optimizations as well as some changes to the voxel logic.

The project was written in the now depricated Unity Script.

Changes to Nihilist Dev’s implementation:

  • Voxel data structure modifications to appropriately calculate flame propagation.
  • Minor addition of ‘voxel intelligence’ in order to determine the probability whether a voxelized surface area of a mesh should ignite or not.
  • Culling voxels within the mesh. This optimization was appropriate in the given scope, since there was no need to simulate flame propagation other than on the surface of the mesh.
