Project
Sponza

//A real-time render of the Sponza scene using an OpenGl graphics engine.

The purpose of this project was to take a deep dive into the OpenGL graphics pipeline. This 3D-scene implements various graphics programming techniques, in order to optimize and create visually aesthetic effects.

Graphics Programming Techniques

  • Deferred Rendering.
  • Shadow Mapping.
  • GPU profiling using NVIDIA's own profiling tool NSight.
